# Docker + WordPress

Quick setup for WordPress environment with Docker 

### Prerequisites
* [Docker]
* [Docker Compose]

## Usage

#### Run Docker Compose

`docker-compose up -d`

Open `localhost:8000` in the browser

#### Shutdown and cleanup

`docker-compose down`



[Docker]: https://docs.docker.com/install/
[Docker Compose]: https://docs.docker.com/compose/install/
